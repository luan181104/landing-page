Project Introduction

Claim Request is a system that supports employees to submit requests for overtime (OT) pay quickly, transparently and conveniently. With the goal of optimizing the OT management process, Claim Request helps employees easily create requests after each overtime shift, while supporting the HR and accounting departments in reviewing, processing and synthesizing accurate data.

Main Features

Staff

  • 1 View dashboard and OT date in calendar
  • 2 Create claim OT request
  • 3 View claim
  • 4 Update claim with status draft

Approver (PM)

  • 1 Manage project
  • 2 Approve claim request
  • 3 Return claim request
  • 4 Reject claim request

Finance

  • 1 Pay claim request
  • 2 Reject claim request
  • 3 Download paid claim request

Admin

  • 1 Manage projects
  • 2 Manage staffs
  • 3 View overall statistical report
